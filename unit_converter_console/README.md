# Unit Converter Console

## Description
Converter app for temperature, length, and weight units in a menu-driven interface.

## Concepts Covered
- Functions
- Conditionals
- Switch statements
- Arithmetic formulas
- Input validation

## Build
```bash
gcc main.c -o unit_converter_console
```

## Run
```bash
# Windows
.\unit_converter_console.exe
# Linux/macOS
./unit_converter_console
```
