# Student Gradebook Manager

## Description
Array-based student record system that stores marks, calculates average/grade, and class summary.

## Concepts Covered
- Arrays and structs
- Loops
- Functions
- Conditionals
- Input/Output
- Formatted tables

## Build
```bash
gcc main.c -o student_gradebook_manager
```

## Run
```bash
# Windows
.\student_gradebook_manager.exe
# Linux/macOS
./student_gradebook_manager
```
