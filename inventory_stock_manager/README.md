# Inventory Stock Manager

## Description
Inventory management console app for adding items, restocking, selling, and valuation reporting.

## Concepts Covered
- Structs and arrays
- Functions
- Loops
- Conditionals
- Input validation

## Build
```bash
gcc main.c -o inventory_stock_manager
```

## Run
```bash
# Windows
.\inventory_stock_manager.exe
# Linux/macOS
./inventory_stock_manager
```
