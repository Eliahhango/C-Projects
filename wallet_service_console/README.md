# Wallet Service Console

## Description
Function-based banking mini project with deposit, withdrawal, and balance checking.

## Concepts Covered
- Functions
- Global variables
- Conditionals
- Input/Output
- Menu-driven control

## Build
```bash
gcc main.c -o wallet_service_console
```

## Run
```bash
# Windows
.\wallet_service_console.exe
# Linux/macOS
./wallet_service_console
```
