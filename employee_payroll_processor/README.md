# Employee Payroll Processor

## Description
Payroll calculator that computes overtime, tax, and net salary for multiple employees.

## Concepts Covered
- Functions
- Structs and arrays
- Arithmetic operators
- Conditionals
- Input validation

## Build
```bash
gcc main.c -o employee_payroll_processor
```

## Run
```bash
# Windows
.\employee_payroll_processor.exe
# Linux/macOS
./employee_payroll_processor
```
