# Library Book Manager

## Description
Library catalog manager to add books, borrow/return items, and list status.

## Concepts Covered
- Arrays and structs
- Functions
- Conditionals
- Loops
- Input handling

## Build
```bash
gcc main.c -o library_book_manager
```

## Run
```bash
# Windows
.\library_book_manager.exe
# Linux/macOS
./library_book_manager
```

## Sample Output
```text
=== Library Book Manager ===
1. Add Book
2. List Books
3. Borrow Book
4. Return Book
5. Exit
Choose an option: 2
```
