# Toolbox Calculator

## Description
Menu-driven calculator implementing add, subtract, multiply, and divide operations.

## Concepts Covered
- Switch statements
- Arithmetic operators
- Conditionals
- Input/Output
- Error handling (division by zero)

## Build
```bash
gcc main.c -o toolbox_calculator
```

## Run
```bash
# Windows
.\toolbox_calculator.exe
# Linux/macOS
./toolbox_calculator
```

## Sample Output
```text
=== Toolbox Calculator ===
1. Add
2. Subtract
3. Multiply
4. Divide
5. Exit
Choose an option: 4
Enter two numbers: 12 0
Error: division by zero is not allowed.
```
