# Contact Book Manager

## Description
C program to add, list, search, and delete contacts using an in-memory array.

## Concepts Covered
- Arrays and structs
- String handling (fgets, strcmp, strstr)
- Loops
- Functions
- Input validation
- Menu systems

## Build
```bash
gcc main.c -o contact_book_manager
```

## Run
```bash
# Windows
.\contact_book_manager.exe
# Linux/macOS
./contact_book_manager
```
