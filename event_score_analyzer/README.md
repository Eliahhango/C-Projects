# Event Score Analyzer

## Description
Assignment-style grade calculator for one participant using three round scores.

## Concepts Covered
- Variables and data types
- Arithmetic operators
- Input/Output
- Conditionals (if/else)

## Build
```bash
gcc main.c -o event_score_analyzer
```

## Run
```bash
# Windows
.\event_score_analyzer.exe
# Linux/macOS
./event_score_analyzer
```

## Sample Output
```text
=== Event Score Analyzer ===
Enter participant name: John
Enter scores for 3 rounds: 75 68 80
Participant: John
Total Score: 223.00
Average Score: 74.33
Level: A
```
