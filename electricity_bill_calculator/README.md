# Electricity Bill Calculator

## Description
C billing system that applies slab pricing for residential and commercial consumers.

## Concepts Covered
- Conditionals and nested ranges
- Arithmetic calculations
- Arrays and structs
- Functions
- Menu-driven processing

## Build
```bash
gcc main.c -o electricity_bill_calculator
```

## Run
```bash
# Windows
.\electricity_bill_calculator.exe
# Linux/macOS
./electricity_bill_calculator
```
