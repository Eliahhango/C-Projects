# Parking Fee Calculator

## Description
Program that issues parking tickets, computes fees by vehicle type, and reports revenue.

## Concepts Covered
- Functions
- Conditionals
- Structs and arrays
- Loops
- Formatted output

## Build
```bash
gcc main.c -o parking_fee_calculator
```

## Run
```bash
# Windows
.\parking_fee_calculator.exe
# Linux/macOS
./parking_fee_calculator
```
