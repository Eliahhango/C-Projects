# BMI Health Checker

## Description
C console program that records BMI entries, classifies categories, and prints statistics.

## Concepts Covered
- Variables and data types
- Arithmetic operators
- Conditionals
- Arrays and structs
- Functions
- Menu-driven flow

## Build
```bash
gcc main.c -o bmi_health_checker
```

## Run
```bash
# Windows
.\bmi_health_checker.exe
# Linux/macOS
./bmi_health_checker
```
