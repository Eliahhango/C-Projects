# Quiz Master Game

## Description
Multiple-choice quiz game with scoring and attempt statistics.

## Concepts Covered
- Arrays
- Loops
- Conditionals
- Functions
- Input validation
- Menu systems

## Build
```bash
gcc main.c -o quiz_master_game
```

## Run
```bash
# Windows
.\quiz_master_game.exe
# Linux/macOS
./quiz_master_game
```

## Sample Output
```text
=== Quiz Master Game ===
1. Play Quiz
2. View Score Statistics
3. Exit
Choose an option: 1
Q1: Which language is primarily used for system programming?
Your answer: 2
Quiz complete. You scored 5/5.
```
